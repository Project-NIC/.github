# NIC-Quark — детекторы излучения (нейтронная опора)

Нейтрон — это адрон, **состоящий из кварков**, поэтому **Quark** — это **нейтронный
домен**: общая нейтронная опора для радиационных плат NIC. По правилу «у платы есть имя»
детекторы делятся на три платы:

- **Photon** — гамма + рентген (трубки ГМ за градуированным Pb, K1–K3); гамма/рентген — это **фотоны**.
- **Helion** — нейтронная трубка He³ (ядро гелия-3 называется *гелион*).
- **Gadolin** — нейтронная плата с захватом на Gd (в честь Юхана Гадолина).

Проект на бумаге — *гражданско-научные относительные мониторы*, плотная
синхронизированная по сети сетка, коррелирующая всплески излучения с молниями (TGF, с
**Tesla**). Не новый тип узла: счёты едут в Basic payload **K1–K4 (смещения 27–30)**.
Финальная нейтронная физика живёт (по-чешски) в `quark/detektor-neutronu/`.
