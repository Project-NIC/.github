# NIC-starDust — air-quality front (parked)

The air you breathe, the dust and the radiation around you: a future NIC front for
**air quality** — the durable basics like CO₂, particulate matter (PM) and VOC, with
room for UV and radiation.

It is **parked**. For now those basics fold into the Weather basic station, and
starDust lives only as a reserved node type (`starDust`). When it lands it will be like
every other front — the universal NIC node core plus a sensor front and thin glue,
announcing its type on the bus and adding **zero** new protocol opcodes.

Reserved now, so the day it ships nothing else has to change.
