# NIC-Tesla — приёмник молний / TGF (сферикс)

Назван в честь **Николы Теслы** (высокое напряжение / катушка Теслы). Самодельный
**сфериксный приёмник** — три ортогональные ферритовые рамки + front-end OPA1637 + АЦП
ADS127L14 + **STM32H523** — который детектирует молнии **на краю** (edge-AI
классификация) и отправляет крошечную **фиксированную запись события**, а не сырые
осциллограммы. **Своя плата** → свой фронт (правило: у платы есть имя).

Поскольку все узлы делят сетевые часы с ~120 нс, каждый разряд получает метку времени для
сетевой мультилатерации (TOA по разнесённым станциям), а гамма-вспышку грозы (**TGF**)
можно сопоставить с радиационными платами (**Quark** / **Photon**).

Проект на бумаге: детектирование — твёрдая часть; локализация требует широкой сети.
